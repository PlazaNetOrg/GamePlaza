# GamePlaza
A modern Android game launcher with optional PlazaNet features.

## Overview
GamePlaza is a custom Android launcher designed for gaming enthusiasts. It provides a unified interface to manage and launch Android games while seamlessly integrating with PlazaNet.

## Features

- **Game Library Management**: Allows you to easily add any Android app as a game, while still allow easy access to all other apps
- **Home Launcher**: Set GamePlaza as your default home screen launcher
- **Social Integration**: Connect with PlazaNet for presence tracking and community features
- **Game Shortcuts**: Create home screen shortcuts for quick game access (mostly for GameHub, and some emulators that support shortcuts)
- **Clean UI**: Modern, dark-themed interface based on PlazaNet design principles
