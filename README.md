# GamePlaza
A Launcher for games with a console-like UI

## Features
- PlazaNet integration:
  - Friends Statuses (`Offline`, `Online (On PlazaNet)`, `Online (In GamePlaza)`, `Playing (GameName)`)
  - Posts
